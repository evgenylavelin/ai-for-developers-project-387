# План развития приложения

## Цель проекта

Настроить процесс командной разработки с использованием OpenCode GitHub App и GitHub Actions.

## Возможные фичи

1. Добавить подтверждение бронирования по email.
2. Добавить выбор формата времени.

## Возможные баги для исправления агентом

1. Некорректная валидация email при бронировании.
2. Ошибка отображения времени.

## Задачи для OpenCode

1. Провести triage issue.
2. Предложить техническое решение.
3. Создать PR с исправлением.
4. Доработать PR после ревью.
5. Настроить регулярную проверку Lighthouse через GitHub Actions.
