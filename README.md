# 📅 Call Calendar

Приложение для бронирования временных слотов (аналог Calendly) без регистрации пользователей.

Проект реализуется с использованием подхода Design First, где API-контракт определяет поведение системы и служит единым источником правды для разработки.

---

## 🧭 Описание

Система позволяет владельцу календаря создавать типы событий, а гостям — бронировать доступные временные слоты.

Приложение не требует регистрации или авторизации:
- владелец календаря — фиксированный пользователь
- гость — любой пользователь без аккаунта

---

## 🧠 Архитектурный подход

Проект разрабатывается по принципу **Design First**:

- сначала определяется API-контракт
- затем независимо реализуются frontend и backend
- контракт является единой точкой синхронизации

Это позволяет:
- разделить ответственность между частями системы
- упростить разработку
- снизить связанность компонентов

---

## 🔌 API

API описывается с помощью **TypeSpec**:

- контракт задаёт структуру запросов и ответов
- на его основе может быть сгенерирована OpenAPI-спецификация
- используется как источник правды для всех частей системы

---

## 👥 Роли

### Владелец календаря (Owner)

- создаёт типы событий
- просматривает список всех бронирований

### Гость (Guest)

- просматривает доступные типы событий
- выбирает событие
- бронирует свободный временной слот

---

## 📌 Основные сценарии

### Управление типами событий
- создание типа события
- получение списка типов событий

### Бронирование
- выбор типа события
- просмотр доступных слотов
- создание записи

### Просмотр бронирований
- получение списка всех записей (для владельца)

---

## ⚙️ Бизнес-правила

### Уникальность слотов
Нельзя создать две записи на одно и то же время  
(даже для разных типов событий)

### Окно бронирования
- доступны только ближайшие 14 дней
- слоты формируются от текущей даты

### Доступность
- бронирование возможно только в свободные слоты

## ▶️ Запуск

Сборка TypeSpec-контракта:

```bash
npm run spec:compile
```

Форматирование файлов контракта:

```bash
npm run spec:format
```

Запуск frontend-приложения:

```bash
npm run frontend:dev
```

Сборка frontend-приложения:

```bash
npm run frontend:build
```

Запуск frontend-тестов:

```bash
npm run frontend:test -- --run
```

- изменения централизованы через контракт
